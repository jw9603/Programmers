## 1. 두 수의 합


### 문제 설명


정수 num1과 num2가 주어질 때, num1과 num2의 합을 return하도록 soltuion 함수를 완성해주세요.

### 제한사항
-50,000 ≤ num1 ≤ 50,000
-50,000 ≤ num2 ≤ 50,000

<td>[2, 3, 5]</td>
</tr>

입출력 예 #1
